## Featured work

### [OCPP DebugKit](https://github.com/ocpp-debugkit/ocpp-debugkit)

> Open-source DevTools for debugging OCPP charging sessions.

[![Release](https://img.shields.io/github/v/release/ocpp-debugkit/ocpp-debugkit?display_name=tag&label=release)](https://github.com/ocpp-debugkit/ocpp-debugkit/releases)
[![CI](https://github.com/ocpp-debugkit/ocpp-debugkit/actions/workflows/ci.yml/badge.svg)](https://github.com/ocpp-debugkit/ocpp-debugkit/actions/workflows/ci.yml)
[![License](https://img.shields.io/badge/license-Apache--2.0-blue.svg)](https://github.com/ocpp-debugkit/ocpp-debugkit/blob/main/LICENSE)

OCPP DebugKit helps charging-infrastructure developers diagnose failures and understand what happened during an OCPP session.

- Inspect charging-session traces, timelines, and individual messages
- Detect common authorization, connector, and connectivity failures
- Run predefined trace scenarios
- Generate shareable Markdown reports
- Analyze traces through a browser-based inspector or CLI
- Keep sensitive trace data local when using the web application
